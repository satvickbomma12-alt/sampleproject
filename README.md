# My First Website

A simple portfolio-style static website inspired by the example page at `https://yashwanth.up.railway.app/about.html`.

## Files

- `index.html` — Landing page with hero, skills, and a contact section.
- `about.html` — About page with work history, education, and contact details.
- `styles.css` — Shared styling for layout, glassmorphism, and responsive design.

## Run locally

1. Open the folder in VS Code.
2. Open `index.html` in your browser.

For best results, use Live Server or a simple HTTP server:

```powershell
cd C:\Users\satvi\my-first-website
python -m http.server 5500
```

Then visit `http://localhost:5500`.
